WebICE User Manual

 

Web-based Interspecies Correlation Estimation (Web-ICE) for Acute Toxicity: User Manual (24 pp, 1.2 MB, About PDF)

Web-based Interspecies Correlation Estimation (Web-ICE) estimates the acute toxicity of a chemical to a species, genus, or family from the known toxicity of the chemical to a surrogate species. 

This manual provides step-by-step instructions for using Web-ICE, as well as information on the databases, model development, model validation, and user guidance on model selection and interpretation.
